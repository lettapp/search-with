# Lett Search With

Quickly search for `selected text` with any search engine. It works similar to Chrome's built-in feature (Search Google for) but with the flexibility of defining your own search engines.

The extension provides the ability to:
1. Add, remove or edit unlimited number of search engines.
2. Select the order in which they appear in the context menu.

In addition, you can add from a common list of search engines including PubMed, Spotify, Reddit and Urban Dictionary among others. Newly added engines are instantly updated in the right-click context menu.

The extension is available on the [Chrome Web Store][cws].
